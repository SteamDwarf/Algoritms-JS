Примеры функций

  1. ln( x + sqrt( x^2 + 1.21 )) / x^3
  2. ( 1 / x^2 ) * arcsin( x / 9 )
  3. x * e^(-x)
  4. sqrt( sin(x) ) * cos(0.5 * x)
  5. 1 / (10 + 6 * sin(x + e))
  6. cos(x) / ( x^2 + 1 )
  7. cos(x) / ( sin(x) + ( 1 + sin(x) ))
  8. x^5 / (sqrt(0.36 + x^2))^5
  9. 1 / (0.5 * sin(x) + 3 * cos(x))^2

Введите ключевую информацию

Указания

  1. Десятичные дроби необходимо вводить не через запятую, а через точку
  2. На данный момент программа понимает такие математические константы и методы, как: ln, cos, sin, tg, sqrt, arctg, arccos, arcsin, e, pi
  3. При вводе своей функции обязательно перед tg, cos, sin ставить пробел
  4. При расчете методом Симпсона не обязательно вводить количество разбиений, так как был создан метод, который подсчитывал оптимальное количество разбиений
  5. При расчете методом Монте-Карло обязательно ввести количество точек
